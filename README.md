Internship Fraud Detection System

📌 Overview
This project detects whether an internship message is Fraud or Legit using text or image input.
It also extracts important details like stipend and fee.

🎯 Features
1.Detect fraud in internship messages
2.Supports text and image input
3.Extracts stipend and fee
4.Uses OCR for image processing
5.Shows result with confidence

🛠️ Technologies Used
-Python, Flask
-HTML, CSS
-SQLite
-Scikit-learn
-Tesseract OCR

⚙️ How to Run
Install requirements
Run:
python app.py
Open:
http://localhost:5000

📁 Files
app.py – main backend
templates/ – HTML pages
static/ – CSS
model.pkl – ML model
